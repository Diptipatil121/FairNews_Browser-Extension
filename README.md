# FairNews Analyzer - Chrome Extension

AI-powered browser extension that analyzes political bias in news articles using NLP pipeline.

## Features
- URL based scraping
- Coreference resolution
- Bias scoring
- Aggregate bias label

## Tech Stack
- FastAPI backend
- Chrome Extension (Manifest v3)
- Cloudflare tunnel

## How to Install
1. Download repo
2. Go to chrome://extensions
3. Enable developer mode
4. Load unpacked
